# **App Name**: Employee Payout Tracker

## Core Features:

- Payroll Period Management: Define and manage pay periods with clear start, end, and pay dates. Allows finalization to prevent further edits for a payroll run.
- Automated Earnings Calculation: Automatically calculate employee earnings based on imported route tracker data (miles, stops, driver/helper roles).
- Custom Payout Adjustments: Manually add or modify 'other earnings' and 'deductions' for each employee to tailor paychecks.
- Paystub Generation & Preview: Generate and preview detailed paystubs for individual employees, including a breakdown of earnings and deductions.
- Comprehensive Payroll Summary: View an overall summary of the payroll run, including total gross pay, deductions, and net pay across all employees.
- Data Export Capabilities: Export the payroll register data to a CSV file for reporting and external record-keeping.
- AI-Powered Note Assistant: An AI tool that assists in summarizing extensive or complex employee-specific notes, helping to extract key information and flag relevant points for review.

## Style Guidelines:

- Primary color: Professional blue (#4461B5) to convey trust and reliability, suitable for main interactive elements.
- Background color: A very light, subtle blue-tinted white (#F1F4F9) for a clean and unobtrusive interface.
- Accent color: A deep, rich indigo-purple (#512999) used sparingly for strong calls to action or highlights to ensure distinctiveness.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly legible characteristics across various text densities, fitting for a data-centric application.
- Utilize modern, clean line icons (like Lucide icons implicitly used in the example) to maintain a minimalist and professional aesthetic.
- Implement a responsive, card-based grid layout with ample spacing and clear visual hierarchy for easy data scanning and readability, particularly in tables.
- Incorporate subtle transitions and animations for user interactions, such as dialog opening/closing or data updates, to enhance user experience without distraction.
